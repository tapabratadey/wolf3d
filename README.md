# wolf3d2.0

Steps to run the Wolf3d-</br>

git clone https://github.com/tapabratadey/wolf3d2.0.git</br>
cd wolf3d2.0</br>
cd libft && make re && cd ..</br>
cd minilibx_macos_sierra && make re && cd ..</br>
make re</br>
./wolf3d map</br>

Movements:</br>
Up: W</br>
Down: S</br>
Right: D</br>
Left: A</br>
